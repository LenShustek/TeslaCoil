This repository contains, in separate directories, design information about
 - a 26" Tesla coil
 - a music-playing modulator for up to four Tesla Coils "singing" in unison
 
 For some videos of the modulator directing a duet with my Tesla coil and 
 a OneTesla (https://onetesla.com/) kit coil, see this YouTube playlist:
 https://www.youtube.com/playlist?list=PLsyKadpUj0KD5guph3T1-Ugx1_Yjd8lQO
